# dimas-prasetyo

Next.js personal site for `dimasboim/dimas-prasetyo`, built for deployment on Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

This repository is ready for Vercel deployment. Connect the GitHub repository to Vercel and deploy the `main` branch.

Vercel will automatically detect the Next.js project.
